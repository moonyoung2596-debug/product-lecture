# Lotto Number Generator

## Overview

This is a simple web application that generates random lottery numbers.

## Project Outline

### Style and Design
*   **Layout:** A centered, single-column layout.
*   **Colors:** A modern color palette with a dark background and vibrant text.
*   **Typography:** Clear and readable fonts.
*   **Effects:** Subtle animations on button clicks and number generation.

### Features
*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers in a clear and visually appealing way.
*   A "Generate" button to trigger the number generation.
